---------------------------Back End----------------------------
Necessário:
	Java 8 ou superior;
	Maven configurado;

Para rodar aplicação: mvn spring-boot:run .


---------------------------Front End----------------------------
Necessário: 
	NodeJs
	Angular 4+

Dentro do diretorio raiz em um terminal digitar: npm install, para baixar as dependências.
Para rodar a aplicação: ng serve
